Vocab-Made-Easy

Java Application made to get a list of definitions given the words.

View project on GitHub

Inspiration

I came up with this idea when I was in 12th grade physics class and I kept getting pounded with a list of vocab I had to do. I was so tired of having to go into the book and find where it was and then writing it down. I would like to see kids in Highschool get to use this program to make their life a lot easier and cut their vocab homework time.

What It Does

The Backend of the program parses the data from dictionary.com and then uses a couple formatting functions to get it to the user how he/she wants it.

What I Made It With

I made the Graphical User Interface(GUI) With Netbeans, and then coded the actual project inside of Eclipse and IntelliJ. I am the only contributor at this time, Andrew Corp @corpandrew
